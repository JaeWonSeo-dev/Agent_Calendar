# Agent Calendar UI Layout Draft

## Main workspace

### Left panel (70%)
Calendar workspace:
- month / week / day switcher
- event grid / calendar view
- event create button
- event detail modal
- user color indicators on events

### Right panel (30%)
LLM chatbot panel:
- persistent conversation thread
- user message input
- quick prompts:
  - What is on my schedule this week?
  - What is important tomorrow?
  - Did I miss any recurring events this month?
- referenced events / answer metadata

## First-login onboarding
- step 1: choose profile image
- step 2: enter nickname
- step 3: enter birthday
- step 4: choose preferred event color
- step 5: complete and enter workspace
